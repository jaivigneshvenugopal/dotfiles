# Deep Dive


## Vim

### Open, Save, Exit etc
To open a file:
```
vim file.txt
vim              # just opens nerdtree
```

To save a file:
```
:w # saves file
:x # saves and exits
```

To exit a file:
```
:q  # exits a file
:q! # exit without saving a file
```

To switch tabs:
```
ctrl h/l
```
